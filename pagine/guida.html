Guida all'Installazione

Segui questi passaggi per configurare correttamente la libreria 3cia.h nel tuo progetto C.

1. Scarica la libreria

Ottieni il file 3cia.h dalla sezione download della Home. Questo file contiene tutte le funzioni necessarie.

2. Prepara la cartella del progetto

Per un corretto funzionamento, devi organizzare i file in questo modo:

  • Crea una cartella per il tuo progetto (es. "MioProgetto").
  • Copia il file 3cia.h all'interno di questa cartella.
  • Crea il tuo file sorgente C es: Esercizio.c nella stessa cartella.

3. Importazione nel codice

Apri il tuo file .c e aggiungi la dichiarazione di inclusione della libreria in alto.

Nota importante: Trattandosi di una libreria locale, usa le virgolette "3cia.h" e non le parentesi angolari X <Libreria.h> XRicapitolando:

// Corretto: cerca il file nella cartella del progetto
#include "3cia.h"

// Errato: cercherebbe tra le librerie di sistema
// #include <3cia.h>
